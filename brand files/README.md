# Brand Files

Drop client-provided brand assets here. Claude Code wires them up after they land.

## Folders

- `logo/` — primary wordmark, icon mark, monogram, svg & png variants
- `photos/` — studio interior, treatment shots, Jen headshot, product stills

## Naming

Prefer: `logo-wordmark.svg`, `logo-icon.svg`, `studio-hero.jpg`, `jen-headshot.jpg`, `treatment-sugaring-01.jpg`, etc.

## Current placeholders

Every stock photo used on the site is tagged with
`{/* TODO: replace with real studio photo */}` so we can find and swap them.
